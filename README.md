# Latte_1 — 라테일 패션 시뮬레이터

라테일 장비 아이템을 조합해 캐릭터가 옷을 입은 모습을 렌더링하는 데스크톱 시뮬레이터 입니다!

## 다운로드 / 실행

빌드된 실행 파일은 이 저장소의 [**Releases**](../../releases) 에서 zip으로 받아주세요!

1. 최신 릴리스의 `Latte_1_v1.0.0.zip` 다운로드 후 압축 해제.
2. 폴더 안의 `Latte_1.exe` 실행. **`Latte_1.exe`와 `_internal/` 폴더는 반드시 같은 위치에 함께 있어야 합니다!** (분리하면 실행되지 않음).
3. 첫 실행 시 라테일 클라이언트 폴더를 물어보면 LaTale이 설치된 경로를 선택해주시면 됩니다!
